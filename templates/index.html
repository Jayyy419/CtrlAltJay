  • AdaptEd (Work in Progress)

    App Development

    Description: AdaptEd's name comes from Adaptive Education, it signifies our vision as we want to make education adaptable and in turn, equitably fair for all students. How we achieve this is with AdaptEd's unique selling point, users upload teaching material to a database and once the database is well populated enough, they will be able to generate customised worksheets for individual students, pertaining to their proficiency level in a certain subject. AdaptEd utilises a RAG system to break down, load and store teaching material as document embeddings in a vector database, and when the user selects 'Generate customised worksheets', they are prompted to configure proficiency level (and various other factors) for each student. Once done, the system queries the database for texts within the database that are relevant to each student's parameters and pulls information to create a worksheet (and corresponding answer key) for the students to work on. This allows students of different pacing to be able to practice exercises and keep up with the pace of the education.


    Project Start Date: June 2024


    Project End Date: Ongoing


    Role: Frontend Developer


    Deliverable: Educational app to provide adaptive learning resources.


    Technologies Used: Node.js, React, MySQL, OpenAI API


    Challenges Faced: Ensuring real-time updates and scalability.


    GitHub Repository: https://github.com/Jayyy419/adapted


    Future Improvements: Adding more interactive features and parameters for individual students.
